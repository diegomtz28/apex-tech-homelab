
# Naming Standards

## Computer Names
Format: ROLE## (no spaces)

Examples:
- DC01 = Domain Controller
- FS01 = File Server
- WS01 = Workstation
- LOG01 = Logging Server
- HD01 = Helpdesk server

## User Accounts
- Standard users: first.last or firstinitiallast
- Admin accounts: admin-first.last (or first.last-admin)
- Service accounts: svc-purpose (svc_backup, svc_wef, svc_sql)

## Why this matters
- Easier troubleshooting
- Faster inventory understanding
- Cleaner automation targeting
